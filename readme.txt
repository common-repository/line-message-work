== Description ==

WordPressの更新をLINE公式アカウントに通知します。
https://wp-line.work/

== Installation ==

このプラグインは、WordPressダッシュボードから直接インストールできます。

1. [ダッシュボード] >[プラグイン]に遷移
2. "LINE MESSAGE WORK" を検索
3. "LINE MESSAGE WORK" プラグインの横にある "今すぐインストール" をクリックします。
4. プラグインをアクティブにします。
5. [ダッシュボード] > [設定] > [WP LINE Account] で、LINEの [チャネルアクセストークン] と [チャネルシークレット] を設定。
その他は設定はhttps://wp-line.work/2020/06/%e3%83%97%e3%83%a9%e3%82%b0%e3%82%a4%e3%83%b3%e8%a8%ad%e5%ae%9a%e3%81%ae%e4%bb%95%e6%96%b9/　で確認してください。
